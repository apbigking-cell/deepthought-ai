## 启动步骤

### 第一步：获取 DeepSeek API Key
去 https://platform.deepseek.com/api_keys 注册并创建 API Key
新用户送 ¥10 额度，足够聊很久很久

### 第二步：配置
编辑 `.env`，把 `sk-your-api-key-here` 换成你的真实 Key

### 第三步：启动
双击 **start.bat**，会自动启动 Web 管理面板

浏览器打开 http://localhost:3000，选择人格「小语」开始聊天

### 第四步：接入微信（可选）
在终端输入 `/weixin`，扫描二维码登录微信 Bot
微信 Bot 是服务端 API，部署到云服务器后关电脑也能用

## 云端部署（关电脑也能用）

### 🚀 方案一：Railway（推荐，免费）

Railway 每月送 $5 额度，跑一个聊天机器人完全够用。

#### 第 1 步：注册 Railway
打开 https://railway.app → 用 GitHub 账号登录

#### 第 2 步：新建项目
点击 **New Project** → 选择 **Deploy from GitHub repo**
→ 如果你还没把代码传到 GitHub，选 **Empty Project**

#### 第 3 步：上传代码（两种方式）

**方式 A：用 GitHub（推荐）**
```bash
# 先把代码上传到你的 GitHub 仓库
# 然后在 Railway 连接该仓库，自动部署
```

**方式 B：用 Railway CLI**
```bash
# 安装 Railway CLI
npm i -g @railway/cli

# 登录并部署
railway login
cd D:\workbuddy\DeepThought
railway link          # 关联到你的 Railway 项目
railway up            # 上传并部署
```

#### 第 4 步：设置环境变量
在 Railway 项目面板 → **Variables** 添加：

| 变量名 | 值 |
|--------|-----|
| `DEEPSEEK_API_KEY` | `sk-你的真实Key` |
| `DEEPSEEK_BASE_URL` | `https://api.deepseek.com` |
| `DEEPSEEK_MODEL` | `deepseek-chat` |
| `WEIXIN_BOT_TOKEN` | 扫码获取（先留空，部署后再配） |
| `LOCATION_CITY` | `北京` |

#### 第 5 步：获取公网地址
Railway 会自动分配一个 `xxx.railway.app` 域名

### 🐳 方案二：Docker 部署到任意服务器

项目已内置 `Dockerfile` 和 `docker-compose.yml`：

```bash
# 构建并启动
docker compose up -d

# 或者手动构建
docker build -t deepthought .
docker run -d -p 3000:3000 \
  -e DEEPSEEK_API_KEY=sk-你的key \
  deepthought
```

适用场景：你自己的云服务器、阿里云、腾讯云等。

## 部署后接入微信

云端部署完成后，获取微信 Token：

1. 通过 Railway 的 Web Terminal 或 SSH 登录
2. 执行命令：
   ```bash
   node src/index.js
   ```
3. 在终端输入 `/weixin`
4. 用微信扫码
5. 复制显示的 Token
6. 回到 Railway Variables 设置 `WEIXIN_BOT_TOKEN=你的Token`
7. 重启部署

之后手机微信给机器人发消息，AI 就会以你设定的人格回复。
